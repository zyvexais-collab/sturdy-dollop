BEST INSTALL METHOD
1. Upload the folder contents to GitHub Pages, Netlify Drop, or any simple static host.
2. Open the site in Safari on iPhone.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open the installed app once and tap through Home, Log, Attack, and Daily so everything caches.

DIRECT ICONS
If little one wants separate Home Screen launch icons, also open:
- log.html
- attack.html
- daily.html
and use Share -> Add to Home Screen on each one.
